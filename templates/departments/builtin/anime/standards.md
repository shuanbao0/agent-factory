# 动漫制作部执行标准

## 标准工作循环

```
1. 分析市场趋势和平台热门题材
2. 策划选题和剧本大纲
3. 完成分镜和角色设定
4. 制作动画（原画→上色→合成）
5. 后期处理（音效、配乐、字幕）
6. 发布并追踪数据（完播率、互动率）
```

## 制作原则

1. **制作规范** — 遵循动画制作流水线标准（分镜→原画→上色→后期），不跳步骤
2. **风格统一** — 同一作品内角色设计、场景风格、色彩方案保持一致
3. **进度协调** — 各环节产出必须按时交付，不阻塞下游
4. **数据选题** — 题材选择基于平台数据（热搜、完播率、评论趋势），不拍脑袋
5. **短视频优化** — 竖屏 9:16、1-3 分钟、前 3 秒留住观众、高完播率

## 项目知识文件

每部作品维护 `ANIME-GUIDE.md`：
- 作品世界观和角色设定表
- 视觉风格规范（色板、线条风格、特效标准）
- 制作进度和各集状态
- 平台发布策略和数据基线
- 音效和配乐资源索引

## 质量标准

- 角色设计一致性：同一角色在不同场景中外观比例一致
- 动画流畅度：关键动作不低于 12fps，特效不低于 24fps
- 剧本逻辑完整：无情节断裂、无前后矛盾
- 音画同步：配音、音效、画面时间点准确对齐
- 字幕无错误：文字内容准确，时间轴精准

## 退出标准

每集/每期完成时验证：
- 分镜与剧本一致（导演确认）
- 角色和场景符合设定表
- 后期处理完成（音效、配乐、字幕）
- 平台规格检查通过（分辨率、时长、文件格式）
- 制作进度表已更新

## 协作规则

- 剧本确认后才能开始分镜，分镜确认后才能开始原画
- 角色设计变更需通知所有已开工的环节
- 声优/配乐的录制与动画制作并行，但合成在动画完成后
- 批量制作同系列作品时复用角色/场景资源，减少重复劳动
- 市场数据分析结论同步给选题策划，形成数据驱动选题闭环
