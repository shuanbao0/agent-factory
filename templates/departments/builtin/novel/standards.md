# 小说创作部执行标准

## 创作流程
- 章节写作必须按大纲顺序执行，不跳章
- worldbuilding 和 character 任务必须在 writing 任务之前完成
- 每章完成后需与大纲核对一致性

## 质量侧重
- 评审侧重故事性和文学性，而非格式规范
- 角色设定、世界观、情节线必须跨章节一致
- 同一作品内文风、叙事视角保持统一
- 避免流水账式叙述，注重场景描写和对话质量

## 协作规则
- Writer 在收到 Worldbuilder 和 Character Designer 的产出后才能开始写作
- Plot Architect 的大纲修改需通知所有相关 Writer
- Continuity Manager 有权要求任何章节返工以保持连贯性
- Style Editor 的修改建议优先级高于进度要求

## 特殊规则
- 创作类任务允许更长的执行周期，不因短期无产出判定失败
- 灵感类任务（worldbuilding、character）允许多次迭代
- 最终产出字数不低于大纲预设的 80%
