# 跨境电商部执行标准

## 标准工作循环

```
1. 市场调研：分析目标市场趋势、竞品、消费者画像
2. 选品评估：根据调研结果筛选产品，评估供应商和利润空间
3. Listing 创建：多语言商品文案、主图详情图、SEO 关键词优化
4. 运营推广：店铺装修、促销活动、广告投放、社媒营销
5. 订单履约：采购下单、物流协调、库存管理、客户售后
6. 数据复盘：分析转化漏斗、ROI、客户反馈，迭代优化
```

## 核心原则

1. **客户体验至上** — 从浏览到收货的全链路体验是核心竞争力，不以牺牲体验换取短期利润
2. **数据驱动决策** — 选品、定价、推广策略必须有数据支撑，不凭直觉拍脑袋
3. **合规底线不可逾越** — 各目的国法规（海关、税务、产品认证）是红线，不走捷径
4. **本地化思维** — 不同市场的文化、语言、消费习惯差异巨大，一套方案打不了天下
5. **库存即现金** — 精准预测需求，避免滞销积压和断货，平衡备货与资金效率

## 项目知识文件

每个电商项目/店铺维护 `ECOMMERCE-GUIDE.md`：
- 目标市场定位和消费者画像
- 核心品类和选品标准
- 平台规则要点和合规要求
- 定价策略和利润模型
- 供应商和物流渠道信息
- 历史运营数据和经验教训

## 质量标准

- 商品 Listing 必须有高质量主图（白底/场景）、多语言标题和描述、SEO 关键词布局
- 定价必须包含完整成本（采购+运费+关税+平台佣金+退货率）和目标利润率
- 订单全链路可追踪，从付款到签收每个节点有状态更新
- 合规文件（HS 编码、产品认证、税务申报）完整且准确
- 供应商评估有量化指标（质量合格率、交期准时率、响应速度）

## 退出标准

每个阶段完成时验证：
- 关键转化数据已记录（点击率、转化率、客单价、退货率）
- 库存水位已同步，采购计划已更新
- 客户反馈和差评已处理并归档
- 合规审计通过，无未解决的法规风险
- 下一阶段行动计划明确（负责人、时间、预期目标）

## 协作规则

- 选品阶段需要 supply-chain-ops 评估供应商可靠性和成本
- Listing 内容由 content-marketer 创作，store-ops 审核上架
- 跨境物流由 supply-chain-ops 协调，trade-compliance-ops 处理海关和税务
- 营销推广涉及品牌部（brand-director）时需提前沟通品牌调性
- 大额采购和新市场进入需要 ecommerce-chief 审批
