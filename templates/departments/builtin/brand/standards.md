# 品牌传播部执行标准

## 标准工作循环

```
1. 分析品牌定位和目标受众
2. 策划内容方案（选题、渠道、传播策略）
3. 创作内容（文案、视觉、视频脚本）
4. 审核：品牌一致性 + 合规性 + 事实准确性
5. 发布并监控传播效果
6. 复盘数据，沉淀方法论
```

## 品牌原则

1. **品牌一致** — 所有对外内容必须符合品牌调性和视觉规范
2. **受众导向** — 内容创作以目标受众需求为核心，不自嗨
3. **创意与合规并重** — 创意表达不能违反法规和品牌底线
4. **SEO 思维** — 内容创作同步考虑搜索引擎优化
5. **长期主义** — 品牌建设是长期投入，不追求一时的流量爆发

## 项目知识文件

每个品牌项目维护 `BRAND-GUIDE.md`：
- 品牌定位和核心价值主张
- 视觉规范（颜色、字体、Logo 使用规则）
- 内容调性指南（语气、措辞、禁忌词）
- 各渠道内容策略和发布节奏
- 传播效果数据基线

## 质量标准

- 所有对外内容经过事实核查（数据、引用、产品描述）
- 视觉素材符合品牌规范（颜色、排版、Logo 放置）
- 文案无错别字、无语病、无歧义
- SEO 关键词自然融入，不影响可读性
- 竞品提及客观中立，不贬低不攻击

## 退出标准

每份内容发布前验证：
- 品牌调性一致（对照品牌指南核查）
- 事实准确无误
- 法务合规通过（涉及法规时需法务审核）
- 视觉质量达标
- 品牌知识文件已更新

## 协作规则

- PR 稿件涉及产品功能描述时需 PM 确认
- 视觉素材与设计部门协同，确保品牌一致
- 危机公关响应需法务部门配合
- 传播数据定期同步给运营部和 CEO
