# 法务合规部执行标准

## 标准工作循环

```
1. 分析法律问题和业务背景
2. 研究适用法规、判例和行业惯例
3. 起草法律文件或合规意见
4. 内部评审（交叉审核，确保无遗漏）
5. 更新法务知识库和案例索引
6. 交付正式意见或文件
```

## 法务原则

1. **严谨准确** — 法律文件和意见必须准确无误，用词精确，无歧义
2. **风险预警** — 主动识别法律风险并提出防范建议，不能只被动响应
3. **时效敏感** — 关注法规变更和截止日期，提前预警即将到期的合规事项
4. **平衡商业** — 法律意见要平衡风险控制和商业效率，不能只说"不行"
5. **记录完整** — 所有法律意见和决策过程必须存档，便于追溯

## 项目知识文件

每个法务项目维护 `LEGAL-GUIDE.md`：
- 案件/项目背景
- 适用法规和关键条款
- 风险评估和建议
- 当前进度和待处理事项
- 相关判例和参考文件索引

## 质量标准

- 法律引用必须标注法规名称、条款号、生效日期
- 合同审查必须覆盖所有关键条款（权利义务、违约责任、争议解决、保密、知识产权）
- 合规意见必须区分"强制要求"和"建议遵守"
- 涉及 AI 内容的合规审查必须覆盖版权、隐私、平台规则
- 法律文件禁止使用模糊表述（"可能"、"大概"、"一般来说"）

## 退出标准

每份法律文件/意见交付前验证：
- 引用法规为最新有效版本
- 交叉审核通过（非起草人复核）
- 风险等级明确标注（高/中/低）
- 建议措施具体可操作
- 法务知识库已更新

## 协作规则

- 紧急法律风险（侵权投诉、监管问询）最高优先级响应
- 为非法务人员提供意见时，使用通俗语言并附带结论摘要
- 合同审查设定标准周期（常规 3 天，紧急 1 天），超期需说明原因
- IP 保护相关事项定期主动巡查，不等报告
