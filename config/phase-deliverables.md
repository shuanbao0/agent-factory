# 阶段交付物模板 — 通用模板

> 项目进入每个阶段时，系统自动生成该阶段的交付物骨架文件。
> 部门可通过 `config/departments/{id}/config.json` 的 `workflow.phaseDeliverables` 覆盖。
> Agent 填写内容，完成后即满足该阶段的交付物要求。

## UNIVERSAL

#### docs/requirements.md
