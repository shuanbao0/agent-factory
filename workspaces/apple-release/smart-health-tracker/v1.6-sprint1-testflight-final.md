## 3. Developer Portal 操作清单

> 提供给 CEO 的人工操作步骤

### 3.1 App Store Connect 分发准备

#### 步骤1: 创建 App Store Connect 应用（如尚未创建）

1. 登录 [App Store Connect](https://appstoreconnect.apple.com)
2. 点击「我的 App」→「+」→「新建 App」
3. 填写信息：
   - 平台: iOS
   - 名称: Smart Health Tracker
   - 主要语言: 简体中文
   - 套装 ID: `com.smarthealth.SmartHealthTracker`
   - SKU: `smart-health-tracker-v160`
4. 点击「创建」

#### 步骤2: 配置 App 信息

1. 在「App Store」标签页填写：
   - 副标题: 你的智能健康伙伴
   - 类别: 健康与健身 / 生活风格
   - 内容版权: © 2026 Smart Health Inc.
2. 在「价格与定价」设置价格（建议免费）
3. 在「隐私」标签页填写隐私政策 URL

#### 步骤3: 上传构建版本

1. Xcode 中选择「Product」→「Archive」
2. 完成后在 Organizer 点击「Distribute App」
3. 选择「App Store Connect」→「Upload」
4. 完成签名和上传

#### 步骤4: TestFlight 分发

1. 在 App Store Connect 选择构建版本
2. 点击「测试」标签
3. 添加内部测试员或创建外部测试组
4. 提交构建版本供 Apple 审核（外部测试需要）

### 3.2 HealthKit Entitlement 申请（如需要）

⚠️ **重要**: HealthKit entitlement 需要 Apple 单独审批

1. 登录 [Apple Developer Portal](https://developer.apple.com)
2. 进入「Certificates, Identifiers & Profiles」
3. 选择 App ID `com.smarthealth.SmartHealthTracker`
4. 在「Capabilities」中启用「HealthKit」
5. 如需后台交付，启用「HealthKit (Background Delivery）」
6. 保存并生成新的 Provisioning Profile
7. 在 Xcode 中重新签名

### 3.3 注意事项

| 注意事项 | 说明 |
|---------|------|
| **审核时间** | HealthKit entitlement 审核可能需要 1-2 天 |
| **测试范围** | 外部测试需要 Apple 审核（1-2 天） |
| **年龄分级** | HealthKit 应用通常需要 17+ 分级 |
| **隐私政策** | 必须提供有效的隐私政策 URL |
| **截图** | 需要提供所有尺寸的 App Store 截图 |
