# AlternativeTo Submission Guide

## How to Submit

1. Go to https://alternativeto.net/
2. Sign up / Log in (top right)
3. Click your profile → "Add new application"

## Submission Details

### Basic Info
- **Name**: Agent Factory
- **URL**: https://github.com/shuanbao0/agent-factory
- **Description**: Self-contained multi-agent collaboration platform that turns one person into a fully staffed AI-powered organization. Features 64 pre-built role templates (CEO, PM, engineers, designers, testers, marketing, legal, finance, quant trading), a Dashboard UI with pixel-art office visualization, autopilot mode, and built-in OpenClaw engine. Supports 15+ LLM providers. No cloud dependency — runs entirely on your local machine.
- **License**: Open Source (GPL-3.0)
- **Platforms**: macOS, Linux
- **Tags**: AI, Multi-Agent, Automation, Self-Hosted, Open Source, Developer Tools

### Suggest as Alternative To
- **CrewAI** — Agent Factory offers 64 pre-built templates vs CrewAI's define-your-own approach, plus a built-in Dashboard UI
- **AutoGen** — Agent Factory is TypeScript/self-contained vs AutoGen's Python/notebook-based approach
- **MetaGPT** — Both simulate software companies, but Agent Factory covers more departments (64 roles) and has a web Dashboard
- **ChatDev** — Agent Factory has a broader scope (not just software dev) and real-time monitoring UI
