## Medium Pitch (3 paragraphs)

Hi [Editor name],

I'm reaching out about **Agent Factory**, an open-source multi-agent collaboration platform that I think your readers would find interesting. The premise is simple: give one person the operational capacity of an entire company through 64 specialized AI agents that autonomously collaborate.

What makes Agent Factory different from frameworks like CrewAI or AutoGen is that it's fully self-contained. There's no external orchestration service, no cloud dependency -- you run `npx agent-factory start` and get a complete local environment with a Dashboard UI, pixel-art office visualization, task board, message center, and autopilot mode. The CEO agent decomposes high-level goals into tasks, assigns them to the right specialist agents (PM, frontend engineer, designer, QA, etc.), and they execute in parallel, producing real work artifacts.

The platform ships with 64 pre-built role templates organized into 5 departments (Development, Business, Content, Creative, Quant), supports 15+ LLM providers, and features an N x N agent communication matrix with fine-grained permission controls. It's built with Next.js 14, TypeScript (strict mode), and Tailwind CSS. The dark-themed UI includes real-time monitoring of tasks, agent status, logs, and resource usage.

Key features:
- **Autopilot mode**: One-sentence goals to full task execution
- **64 agent templates**: Complete organizational structure from CEO to quant trader
- **Pixel-art office**: Retro visualization showing agent status and departments
- **Self-contained**: Built-in OpenClaw engine, no external dependencies
- **Multi-provider**: Anthropic, OpenAI, DeepSeek, MiniMax, and 10+ more
- **Agent communication**: N x N permission-controlled message system
- **Skill marketplace**: Extend agent capabilities through ClawHub
