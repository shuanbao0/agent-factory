In this demo, you'll see how 64 AI agents -- CEO, engineers, designers, testers, marketers, and more -- autonomously collaborate to deliver real work artifacts.

0:00 - Introduction

=== KEY FEATURES ===
- 64 pre-built agent templates across 5 departments
- Autopilot mode: one-sentence goals to full execution
- Pixel-art office UI for spatial visualization
- Self-contained: no cloud, no external orchestration
- 15+ LLM providers (Anthropic, OpenAI, DeepSeek, MiniMax)
- N x N agent communication with permission controls

Next.js 14 + TypeScript + Tailwind CSS + OpenClaw Engine
